Objectifs

Réaliser un jeu 2D complet avec Unity fonctionnant sous Android :

  • Utilisation de la physique : Collider + Rigidbody
  • Stockage des meilleurs scores via les PlayerPrefs
  • Réalisation des images du jeu via Inkscape
  • Gestion des animations
  • Gestion multi-langage via XML
  • Déploiement sur le playstore
  • Intégration de publicité via Unity Ads

Le jeu

L'objectif du jeu est de faire avancer un spermatozoïde jusqu'à un ovule tout en évitant les parois et les virus. Si vous touchez ces derniers, vous devez recommencer le niveau.


Par défaut, le spermatozoïde tourne sur lui-même. Pour le faire avancer, il faut cliquer sur l'écran afin de changer le côté sur lequel il tourne afin de le faire zigzaguer.

Par defaut, le spermatozoïde tourne sur lui-même.


Au clique, il change de direction.


Plusieurs cliques le font avancer en zigzag.


Vos meilleurs temps sont enregistrés et vous gagnez des médailles si vous êtes assez rapide.


Il existe actuellement deux mondes composés de huit niveaux. Ainsi que trois niveaux de difficulté qui changent la vitesse de rotation du spermatozoïde.

Les différents mondes et niveaux de difficulté se débloque grâce aux médailles obtenues par le joueur.
